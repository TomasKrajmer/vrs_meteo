# vrs_meteo

Meterologická stanica s predpoveďou počasia

Hardvér:
- STM32F303K8TX
- ILI9341TFT
- IKS01A1

Úlohy:
- čítanie, formátovanie a spracovanie dát zo senzorov cez I2C
- ovládanie a nastavovanie aplikácie cez PC komunikácia cez USART
- komunikácia s displejom a dotykovou plochou cez SPI
- grafické rozhranie, vykreslovanie údajov a objektov
